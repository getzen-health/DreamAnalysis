Svapnastra — Architecture Guide

Complete breakdown of how every page, score, ML model, and data flow works in the application. Open each section to learn more.

1. How Data Flows

1
Muse 2 Headband

4 EEG channels (TP9, AF7, AF8, TP10) at 256 Hz via Bluetooth

↓
2
BrainFlow v5.20.1

Reads raw EEG signals from Muse 2 (board ID 38, native Bluetooth)

↓
3
FastAPI Backend (port 8000)

12 ML models process each chunk: sleep, emotion, dream, flow, creativity, memory, drowsiness, cognitive load, attention, stress, lucid dream, meditation

↓
4
WebSocket /ws/eeg-stream @ 4Hz

Sends JSON frames with all 12 model outputs, band powers, signal quality, emotion shifts, and coherence data

↓
5
DeviceProvider (React Context)

Stores latestFrame globally. Every page reads from useDevice() hook. Throttled to 1.5s updates for readability.

↓
6
Pages Render Live Data

Each page reads latestFrame.analysis.* — no fake data. All zeros when not streaming.

2. The 12 ML Models

Every WebSocket frame runs your EEG through all 12 models. Each returns a score (0-1) and a state label.

Sleep Staging

Output: stage (Wake/N1/N2/N3/REM), confidence
How: Delta/theta/alpha ratios + spindle detection

Emotion

Output: emotion (6 types), valence, arousal, stress, focus, relaxation
How: CNN-LSTM or feature-based from band power ratios

Dream Detection

Output: is_dreaming, probability, rem_likelihood, intensity, lucidity
How: REM-like patterns, theta dominance, eye movement

Flow State

Output: in_flow, flow_score
How: High focus + moderate arousal + low stress pattern

Creativity

Output: state, creativity_score
How: Theta-alpha ratio — high theta + moderate alpha = creative

Memory Encoding

Output: state, encoding_score
How: Theta bursts + gamma coupling = memory consolidation

Attention

Output: state, attention_score
How: Prefrontal beta activity. High beta + low theta = focused

Drowsiness

Output: state, drowsiness_index
How: Rising theta + dropping alpha + slow eye movements

Cognitive Load

Output: level, load_index
How: Frontal theta power. High theta + high beta = overload

Stress

Output: level, stress_index
How: High beta/alpha ratio + frontal asymmetry

Meditation

Output: depth (Surface/Light/Deep/Jhana), score
How: Strong alpha coherence + low beta = meditative

Lucid Dream

Output: state, lucidity_score (only during REM)
How: Gamma bursts in frontal regions during REM

3. EEG Band Powers

Raw EEG is decomposed into 5 frequency bands. All values are relative power (0-1).

Alpha8-12 HzCalm focus, relaxation with awareness, eyes-closed rest
Beta12-30 HzActive thinking, problem-solving, anxiety, mental effort
Gamma30-100 HzHigher cognition, memory binding, peak awareness

Neurofeedback (/neurofeedback)

Real-time brain training in 4 phases:

1. Protocol Selection

Choose: Alpha Enhancement, SMR Training, Theta/Beta Ratio, Alpha Asymmetry. Toggle audio.

2. Calibration (30s)

Measures baseline. Relax and breathe normally.

3. Training

Score gauge (0-100). Band powers sent to backend every 1s vs calibrated baseline. Score > threshold = reward (523Hz tone + green flash). Tracks rewards + streaks.

4. Summary

Total rewards, reward rate %, average score, best streak.
