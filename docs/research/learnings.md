| 2026-03-20 | eeg-brain | `preprocess()` in `eeg_processor.py` had zero NaN/inf handling. BrainFlow Bluetooth packet drops inject NaN samples into the EEG stream (1-5% dropout rate on native BT per CLAUDE.md). `scipy.signal.filtfilt` propagates a single NaN to the ENTIRE output array, making every band power, FAA value, emotion probability, and mental state index NaN. Once NaN enters the EMA state in `_predict_features()`, it stays NaN permanently until the classifier is reinitialized — a single dropped packet can poison the entire session. The LGBM paths had partial `np.isfinite` guards but the feature-based heuristic path (the actual live Muse 2 path) had none. | Added `_sanitize_nan()` function that linearly interpolates NaN/inf gaps before filtering (standard EEG approach for short BT drops). Called at the top of `preprocess()` so all downstream code is protected. Returns zeros for all-NaN signals (total disconnection). Fast path skips copy when signal is clean. All 22 IAF tests pass. |
